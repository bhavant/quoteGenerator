# QuoteGenerator

Project to generate Quotes

## Building and running on localhost

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run build-prod
```

To create a development build:

```sh
npm run build-dev
```

## Running and Watching changes

```sh
npm run start
```

## Clean

```sh
npm run clean
```

## Testing

To run unit tests:

```sh
npm test
```
